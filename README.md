# Cancer Awareness & Support

A responsive web application for cancer awareness, education, and community support.

## Features

- **Landing Page**: Hero section with awareness message and feature highlights
- **Contact Form**: Get in touch with name, email, and message fields
- **Daily Inspiration**: Real-time quotes from [Quotable API](https://quotable.io)
- **Responsive Design**: Works on all devices (mobile, tablet, desktop)
- **Simple & Clean**: Built with React and Vite for fast performance

## Quick Start

### Install & Run

```bash
npm install
npm run dev
```

Visit `http://localhost:5173` in your browser.

### Build for Production

```bash
npm run build
```

## Tech Stack

- React 19
- Vite
- Axios
- CSS3
